    C++ Parte 1

    In questa parte sono descritti i tpi incorporati del C++ e i costrutti di base per costruire programmi con essi. Il sottoinsieme C del C++ è presentato insieme al supporto addizionale del linguaggio agli stili tradizionali di programmazione. Saranno anche trattati i costrutti di base per costruire un programma C++ a partire da parti fisiche e logiche.

    Tipi fondamentali

    Il C++ ha un set di tipi fondamentali corrispondenti alle più comuni unità di base di memorizzazione di un computer, e ha i modi più comuni per usarli come dati.

    • Un tipo booleano bool, per i valori logici
    • Tipi carattere char, wchar_t, per i caratteri
    • Tipi interi int, long long, per i valori interi
    • Tipi a virgola mobile double, long double, per i valori con virgola
    • Un tipo per assenza di informazioni void

    I tipi booleani, char e int sono detti tipi integrali.
    I tipi integrali e a virgolia mobile sono detti tipi artimetici.
    Enumerazioni e classi sono detti tipi definiti dall'utente in quanto derivano dai tipi aritmetici.
    I puntatori e riferimenti sono detti tipi incorporati.
    I tipi aritmetici sono forniti di svariate dimensioni al fine di offrire al programmatore la possibilità di scegliere la quantità di spazio di memorizzazione consumato e la precisione e l'intervallo per i calcoli.

    Booleani

    Un booleano può avere valore true oppure false, ed è utilizzato per esprimere i risultati di operazioni logiche. Durante le operazioni, ogni int può essere convertiti in un bool, ma mai viceversa (tranne per i valori 0 ed 1). Gli interi non zero sono convertiti in 1, 0 in false. Ad esempio:

    Listato 1.0

    
                  bool b1 = 7;          // 7!=0, quindi b diventa true.
                  bool b2 {7};          // errore: narrowing

                  int i1 = true;        // i1 diventa 1
                  int i2 {true};        // i1 diventa 1

    Un puntatore può essere implicitamente convertito in bool. Un puntatore non null viene convertito in true; i puntatori con valore nullptr sono convertiti in false:

    Listato 1.1

    
                  void g(int* p)
                  {
                    bool b = p;           // si riduce a true o false
                    bool b2 {p!=nullptr}; // test esplicito con nullptr

                    if (p) {              // equivalente di p!=nullptr (se p è valido)
                      // . . .
                    }
                  }

    Tipi di carattere

    Sono utilizzati molti set di caratteri e codifiche dei set. Per rispecchiare questa varietà il C++ fornisce numerosi tipi:

    • char: Il tipo di carattere predefinito, utilizzato per il testo nei programmi. Di solito è di 8 bit.
    • signed char: Come char, ma in grado di avere un segno: può contenere valori positivi e negativi.
    • unsigned char: Come char, ma con la garanzia di non avere segno.
    • wchar_t: In grado di contenere caratteri più grandi, come ad esempio l'UNICODE.
    • char16_t: In grado di contenere 16 bit, come UTF-16.
    • char32_t: In grado di contenere 32 bit, come UTF-32.

    Quasi universalmente, un char ha 8 bit, e di conseguenza contenere uno tra i 256 diversi valori. Anche se i tipi di set di carattere sono difficili da standardizzare (basti pensare al cirillico e il cinese, ad esempio), si può ragionevolmente presupporre che il set includa le cifre decimali, i 26 caratteri alfabetici della lingua inglese e acluni caratteri di interpunzione di base. Ciascun carattere ha un valore intero nel set di caratteri usato dell'implementazione: ad esempio, il valore di 'b' è 98.

    Tipi interi

    Come char, ciascun tipo intero può avere 3 forme: int "semplice", signed int, unsigned int. Inoltre esistono i seguenti tipi, i quali dipendono dall'intervallo dell'intero che si intende memorizzare:

    • byte
    • short
    • int
    • long
    • long long

    Tipi a virgola mobile

    Per default, un letterale a virgola mobile è di tipo double. Si possono anche usare le notazioni esponenziali, come ad esempio 1.23e-15. Da notare l'assenza degli spazi.

    Tipi di Letterali

    In generale, il tipo di un letterale intero dipende dalla sua forma(come int, float ecc.), dal suo valore e dal suo suffisso. Di seguito sono elencati alcuni dei prefissi e suffissi dei letterali aritmetici:

    Notazione   *fisso Significato Esempio
    0   prefisso ottale 0776
    0x 0X prefisso esadecimale 0xff
    u U suffisso unsigned 12U
    l L suffisso long 2000L
    ll LL suffisso long long 2000LL
    f F suffisso float 10f
    e E infisso virgola mobile 10e-4
    .   infisso virgola mobile 12.3
    '   prefisso char 'c'
    u'   prefisso char16_t u'c'
    U'   prefisso char32_t U'c'
    L'   prefisso wchar_t L'c'
    "   prefisso string "hello"
    R"   prefisso stringa grezza R"(\b)"

    Questi possono essere combinati tra di loro, come ad esempio: 1ULL (1 Unsigned Long Long). Inoltre un utente può definire i propri suffissi, come vedremo in seguito.

    Dichiarazioni

    Prima di poter usare un nome ( identificatore ) in C++, è necessario dichiararlo. Ovvero, il suo tipo deve essere specificato per informare il compilatore del genere di entità a cui il nome si riferisce. Qualsiasi dichiarazione che specifichi un valore è una definizione. Evitando troppe semplificazioni radicali, possiamo considerare che le dichiarazioni siano composte da cinque parti ordinate:

    1. Specificatori prefissi opzionali: static o virtual
    2. Un tipo base: vector<double> o const int
    3. Un dichiaratore che opzionalmente include un nome: [n] oppure *
    4. Specificatori di funzione suffissi opzionali: const o noexcept
    5. Un inizializzatore o corpo della funzione opzionale: ={75.3} o {return x;}

    Tranne le definizioni di funzione e namespace, una dichiarazione termina con un punto e virgola. C'è anche la possibilità di dichiarare più nomi in un'unica dichiarazione. Ecco un paio di esempi:

    Listato 1.2

    
                  char ch;
                  string s;
                  auto count = 1;
                  const double pi {3.141592};
                  extern int error_number;

                  const char* name = "Alessio";
                  const char* season[] = {"sring", "summer", "fall", "winter"};
                  int day(Date* p) { return p->d; }
                  double sqrt(double);
                  template<typename T>T abs(T a) { return a<0 ? -a : a; }

                  constexpr int fac(int n) { return (n<2) ? 1 : n*fac(n-1); }  // possibile valutazione in fase di compilazione
                  constexpr double zz { ii*fac(7) };                              // inizializzazione in fase di compilazione

                  int* p, y;         // int* p, int y: NON int* y
                  int v[10], *pv;    // int v[10], int* pv
